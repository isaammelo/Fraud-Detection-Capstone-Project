# Fraud-Detection-Project-Using-Machine-Learning
Machine learning project for detecting fraudulent online transactions using the IEEE-CIS Fraud Detection dataset.

## Business Problem / Motivation

Credit card fraud is a major issue in the financial industry, leading to significant financial losses and reduced customer trust. Financial institutions need models that can accurately detect fraudulent transactions while minimizing false positives.

This project focuses on building a machine learning model that can identify fraudulent transactions using transaction and identity data.
